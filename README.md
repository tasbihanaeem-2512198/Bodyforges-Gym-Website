🎮 Hangman Game (C Language)

📌 Project Overview
This project is a console-based Hangman game developed using the C programming language. The game challenges the user to guess a hidden word by entering one letter at a time. It is designed to demonstrate fundamental programming concepts such as loops, conditionals, arrays, and string handling.

⚙️ Features
Random word selection
Limited number of attempts
Real-time word progress display
Input validation for letters
Simple and interactive console interface

🛠️ Technologies Used
C Programming Language
Standard Input/Output Library (stdio.h)
String Handling Library (string.h)
Standard Library (stdlib.h, time.h)

🚀 How to Compile and Run
1. Compile the program
gcc hangman.c -o hangman
2. Run the executable
./hangman

🎯 Game Rules
The program selects a hidden word randomly.
You must guess one letter at a time.
Each incorrect guess reduces your remaining attempts.
The game ends when:
You correctly guess the entire word ✔️
Or you run out of attempts ❌
